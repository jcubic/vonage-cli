# Vonage API CLI App

Simple Vonage API CLI app written in Node.js that send SMSes.

## Install

```bash
git clone https://github.com/jcubic/vonage-cli.git
cd vonage-cli
npm install
```

## Usage

```bash
./index.js --phone <PHONE NUMBER> --message <TEXT MESSSAGE>
```
